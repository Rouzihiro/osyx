# Starship

Starship is the shell prompt layer.

It replaces a permanent top bar by putting the important terminal state directly inside the prompt: OS, directory, Git branch, Git status, language runtimes, time, and command status.

Loaded from `zsh/tools.zsh`.

```zsh
eval "$(starship init zsh)"
export STARSHIP_CONFIG="$HOME/.config/starship.toml"
```

The shell starts Starship, then points it at:

```text
.config/starship.toml
```

That file is generated by the theme system, using the template:

```text
flavors/base/starship.toml.j2
        ↓
flavors/generate.py
        ↓
.config/starship.toml
```

## Prompt shape

The prompt is one compact status line plus the command character.

```text
OS → directory → Git branch/status → language runtimes → time
command symbol
```
