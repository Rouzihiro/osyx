# neovim

I barely even use Neovim these days.

Unless I need to edit a file quickly, move fast through a repo, or do something at raw keyboard speed.

Though, this is a modular Neovim setup, well think an old Toyota setup,. Drops in, bootstraps vim-plug on first launch, pulls all plugins, LSPs, and theme, zero manual steps.

## structure

```
init.vim                     → entry point, sources everything in order
core/
  plugins.vim                → vim-plug manifest (themes, helpers, treesitter, coc)
  settings.vim               → shell, encoding, termguicolors, treesitter registration
  keymaps.vim                → all keybinds
plugin_config/
  coc.vim                    → coc extensions, LSP config, completion binds
  telescope.vim              → telescope + file browser setup
  nerdtree.vim               → tree sidebar config
lua/
  theme.lua                  → generated by flavors/generate.py (do not hand-edit)
```

## keybinds

leader is `,`

### core

| key                       | action                                        |
| ------------------------- | --------------------------------------------- |
| `,,`                      | jump to next `<++>` placeholder and change it |
| `ctrl-s`                  | save                                          |
| `ctrl-q`                  | quit without saving                           |
| `ctrl-p`                  | suspend nvim (back to shell)                  |
| `ctrl-z`                  | undo                                          |
| `ctrl-shift-z` / `ctrl-y` | redo                                          |
| `tab` / `shift-tab`       | next / previous buffer                        |
| `ctrl-x`                  | close buffer                                  |
| `S`                       | `:%s//g` quick replace                        |
| `ctrl-a`                  | jump back (after go-to-definition)            |

### splits

| key            | action              |
| -------------- | ------------------- |
| `ctrl-h/j/k/l` | move between splits |

### paths / cwd

| key   | action                          |
| ----- | ------------------------------- |
| `,fa` | echo absolute path              |
| `,ft` | echo filename                   |
| `,fr` | echo path relative to cwd       |
| `,fy` | yank relative path to clipboard |
| `,cd` | lcd to file directory           |
| `,gr` | cd to git root                  |
| `,fg` | echo path relative to git root  |

### telescope

| key      | action                                   |
| -------- | ---------------------------------------- |
| `,f`     | file browser (cwd = current file dir)    |
| `,r`     | live grep (ripgrep)                      |
| `,R`     | grep word under cursor                   |
| `,bl`    | buffer list (sorted by last used)        |
| `ctrl-q` | send selected to quickfix (in telescope) |

### nerdtree

| key  | action                              |
| ---- | ----------------------------------- |
| `,n` | toggle NERDTree (find current file) |
| `,[` | shrink tree                         |
| `,]` | grow tree                           |
| `,=` | equalize splits                     |

### coc / lsp

| key             | action           |
| --------------- | ---------------- |
| `gd` / `ctrl-d` | go to definition |
| `gy`            | type definition  |
| `gi`            | implementation   |
| `gr` / `ctrl-r` | references       |
| `,rn`           | rename           |
| `,ac`           | code actions     |
| `,qf`           | quick fix        |

### ts/js helpers

| key   | action              |
| ----- | ------------------- |
| `,mi` | add missing imports |
| `,oi` | organize imports    |

### completion

| key                 | action                            |
| ------------------- | --------------------------------- |
| `ctrl-space`        | trigger completion                |
| `enter`             | confirm completion                |
| `tab` / `shift-tab` | next / prev item (when menu open) |

### quickfix

| key         | action           |
| ----------- | ---------------- |
| `,qo`       | open             |
| `,qc`       | close            |
| `]q` / `[q` | next / prev item |

### themes / ui

| key   | action                                            |
| ----- | ------------------------------------------------- |
| `,bt` | toggle buffer tabline                             |
| `,th` | cycle colorscheme (runtime only, doesn't persist) |

### misc

| key    | action                                       |
| ------ | -------------------------------------------- |
| `,o`   | toggle spellcheck (en_us)                    |
| `:w!!` | write file as root (suda)                    |
| `,nf`  | prompt for new file (creates dirs if needed) |
