## Specs

- **OS:** Ubuntu
- **DE:** GNOME
- **GTK Theme:** [Tokyonight-Dark-Storm-B-MB](https://www.google.com/url?q=https://github.com/Fausto-Korpsvart/Tokyo-Night-GTK-Theme&sa=D&source=calendar&usd=2&usg=AOvVaw1r35s6Jug6ilmCwulFmhsj)
- **Icon Theme:** [Dracula Icons](https://www.google.com/url?q=https://www.google.com/search?q%3Dhttps://github.com/dracula/gtk/tree/master/kde/icons&sa=D&source=calendar&usd=2&usg=AOvVaw2B9-vidD2Uo7a6_5nnN1rq)
- **Shell Theme:** Tokyonight-Dark-Storm-B-MB
- **Wallpaper:**  [greek mythology](./mytho.png) & [indigo](./indigo.png)
- **Interface Font:** San Francisco Text Medium (12pt)
- **Terminal Font:** [Iosevka Fixed SS18](https://www.google.com/url?q=https://github.com/be5invis/Iosevka&sa=D&source=calendar&usd=2&usg=AOvVaw21KU1cdYjxHPWe-Ad_jIIo) (18pt)
- **Shell:** Zsh & Starship

Extensions (The Essentials)

- **[Blur my Shell](https://www.google.com/url?q=https://extensions.gnome.org/extension/3193/blur-my-shell/&sa=D&source=calendar&usd=2&usg=AOvVaw13IatREOHKw1uke3t0sFu2):** For that frosted glass look on the panel and overview.
- **[OpenBar](https://www.google.com/url?q=https://www.google.com/search?q%3Dhttps://extensions.gnome.org/extension/5073/openbar/&sa=D&source=calendar&usd=2&usg=AOvVaw0J4Xqb0K_QuPtxBv7O9L8A):** This is doing the heavy lifting for the custom top bar styling/colors.
- **[Dash to Dock](https://www.google.com/url?q=https://extensions.gnome.org/extension/307/dash-to-dock/&sa=D&source=calendar&usd=2&usg=AOvVaw1gkbkA5s1VTBCmV8eOC1So):** Positioned at the **Bottom**, opacity set to 0.8.
- **[TilingShell](https://www.google.com/url?q=https://www.google.com/search?q%3Dhttps://extensions.gnome.org/extension/7065/tilingshell/&sa=D&source=calendar&usd=2&usg=AOvVaw0WCSzrbph-5P5CcnQQqW_3):** For the custom window tiling layouts.
- **[Just Perfection](https://www.google.com/url?q=https://extensions.gnome.org/extension/3843/just-perfection/&sa=D&source=calendar&usd=2&usg=AOvVaw3eDvIROsF7M_XxKYIhzhVe):** Used to clean up the UI (hiding search, double-super to app grid, etc.).
- **[Aylur's Widgets](https://www.google.com/url?q=https://github.com/Aylur/gnome-extensions&sa=D&source=calendar&usd=2&usg=AOvVaw3atsjta6QiCqIgXEzunW6G):** Powering the battery bar and dynamic panel tweaks.
- **[Rounded Window Corners](https://www.google.com/url?q=https://extensions.gnome.org/extension/5237/rounded-window-corners/&sa=D&source=calendar&usd=2&usg=AOvVaw35toERGt5JOV6BR1NUd2aN):** For the consistent 12px radius on all apps.
- **[Vitals](https://www.google.com/url?q=https://extensions.gnome.org/extension/1460/vitals/&sa=D&source=calendar&usd=2&usg=AOvVaw1l-knL_RmfcWnSpk4nWseV):** System monitoring in the top bar.
- **[Logo Menu](https://www.google.com/url?q=https://www.google.com/search?q%3Dhttps://extensions.gnome.org/extension/4451/logomenu/&sa=D&source=calendar&usd=2&usg=AOvVaw22WehzQlzVgNfnTjXuLArj):** Replaces the "Activities" button with a custom icon.

Custom Keybinds :

- `Alt + T`: Terminal (GNOME Terminal)
- `Alt + K`: Kitty
- `Alt + G`: Chrome
- `Alt + C:` cursor
- `Alt + F`: Firefox
- `Alt + O`: Obsidian
- `Alt + W`: asryx toggle (my local ASR tool)

Terminal Colors (Dracula-ish)

Using a custom dark palette:

- **Background:** `#000000` (59% transparency)
- **Foreground:** `rgb(255,235,235)`
- **Palette:** Custom mix of Dracula and Tokyo Night colors.
