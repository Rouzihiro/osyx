## Philosophy

A Debian Trixie setup that works, and looks professional.

## The Stack

These have basically been unchanged for years:

- **Compositor:** Hyprland
- **Terminal:** Kitty
- **Multiplexer:** Tmux
- **Shell:** Zsh + Starship
- **Notifications:** Mako
- **Lockscreen:** Hyprlock (or Swaylock for less heat)
- **Clipboard:** Scripts + Wofi for the UI

The top bar has been annihilated to remove unnecessary clutter. Screen real estate is strictly reserved for active logic.
